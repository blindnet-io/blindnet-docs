---
title: Usage Without PCE
sidebar_position: 3
tags: [PRCI, web-components]
---

The PRCI component can be used without an associated Privacy Computation Engine. The example below exposes a PRCI element on a static HTML page.

```html
<body>
  <bldn-priv-request></bldn-priv-request>
  <script src="https://cdn.jsdelivr.net/npm/@blindnet/prci@0.12.3/dist/index.all.min.js" type="module"></script>
</body>
```

When a request is filled out and submitted, the PRCI will emit the [`bldn-priv-request:request-created`](./events#bldn-priv-requestrequest-created) event. You may catch this event and handle it in your system.
