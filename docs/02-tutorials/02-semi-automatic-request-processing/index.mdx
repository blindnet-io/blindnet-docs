# DevKit - Semi-automatic request processing

You may want to manually process certain Privacy Requests.
Data Consumer Interface component allows you to view the Privacy Requests and respond to them.
Privacy Computation Engine is there to recommed you the reponses and make sure you don't forget a rule or a policy.
